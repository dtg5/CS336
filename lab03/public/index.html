Answers for the lab 3 exercises:

  • 3.1

    • Compare and contrast the purposes of Node.js and of Express.js:

      • Node.js is a JavaScript-based framework that was evidently made using Google Chrome's JavaScript V8 Engine. It is used to develop I/O intensive web applications, such as video streaming sites, single-page applications, and other web applications. Express is a Node-based framework which provides a set of features for web and mobile applications. Node.js is like the coding 'engine' (the core programming mechanics), and Express provides a range of development tools for that engine. Node has a module library from which Express is installed, where utility functions can be found for operating systems, file paths, and most of all, a network wrapper for server and client streams.

    • Explain the purpose of the package.json file.

      • A package.json file contains meta data about an app or module. Our particular json package has server, dependency, and author information, as well as some script handling.
